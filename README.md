# Sales Dashboard (Excel Project)

## 📊 Overview

This project is an interactive Sales Dashboard built using Microsoft Excel.

## 🔧 Tools Used

* Excel
* Pivot Tables
* Slicers
* Charts

## 📈 Key Features

* Monthly Sales Trend Analysis
* Sales by Region
* Delivery Performance (On-Time %)
* Interactive Filters (Product, Date, Delivery Status)

## 📌 Key Insights

* Sales show consistent growth, peaking in July
* West region contributes highest revenue
* On-Time Delivery is 51.7%, indicating improvement needed
* Orders increased steadily over time
